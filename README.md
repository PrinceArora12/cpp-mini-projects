# C++ Mini Projects

This repository contains beginner level C++ console projects that demonstrate
basic data structures and programming concepts.

## Projects

### 1. Student Record Management System
Features:
- Add student
- Display all students
- Search student by roll number

Concepts Used:
- Structures
- Arrays
- Functions
- Menu driven program

### 2. Browser History Manager
Features:
- Visit new page
- Go back to previous page
- Display current page

Concepts Used:
- Stack implementation
- Classes
- Arrays
- Object Oriented Programming
