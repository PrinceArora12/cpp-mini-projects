# C++ Mini Projects

This repository contains beginner to intermediate level C++ console projects that demonstrate basic data structures, object-oriented programming, and problem-solving concepts.

## Projects

### 1. Student Record Management System

**Features**

* Add student
* Display all students
* Search student by roll number
* Delete student record
* Save student data to file

**Concepts Used**

* Classes and Objects
* Vectors / Arrays
* File Handling
* Functions
* Menu Driven Program

---

### 2. Browser History Manager

**Features**

* Visit new page
* Go back to previous page
* Go forward to next page
* Display current page
* Show browsing history
* Clear history

**Concepts Used**

* Stack Data Structure
* Classes
* Object Oriented Programming (OOP)
* Menu Driven Program

---

## How to Run

Compile using **g++**

```
g++ main.cpp
./a.out
```
